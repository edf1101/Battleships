This is my main code for running a flask battleships game Includes added extensions: - Guessboards (holds move histories and can show ships sunk) - all board processing is done in python and HTML is just sent a copy of

guessboards (Means HTML website is persistent)

  • Has an entry screen

  • Has difficulty modes

  • Game itself encapsulated in OO
