gui_extensions module¶

These functions below help with the creation and maintenance of a ‘guess_board’. In the extended code the ‘board’ doesn’t get modified after generation, instead the ‘guess_board’ gets modified to state whether a hit/miss/sink was there. This means we have a guess history and can handle the displays python side.
