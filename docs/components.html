components.place_battleships(board: list[list], ships: dict[str, int], algorithm: str = 'simple') → list[list[str]][source]¶

Places all the ships onto the board using a specified placement algorithm :param board: A list of lists representing the board :param ships: A dictionary of ships in the game and their sizes :param algorithm: either ‘simple’, ‘random’ or ‘custom’ :return: A list of lists representing the board, with tiles filled where ships are
