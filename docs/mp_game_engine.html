mp_game_engine module¶

Contains functions needed for a multiplayer game, ie generating attacks etc. Also contains a simple Command line game when module itself is executed

mp_game_engine.generate_attack(board: list[list[str | None]] = None) → tuple[int, int][source]¶

Generates a position to attack via specification random method :param board: input the board to attack so the algorithm knows how large it is. Unittests assume there are no arguments for generate_attack so if we get not argument assume board_size = 10 :return: a tuple coordinate on the grid

mp_game_engine.generate_attack_ext(board: list[list[str | None]], difficulty: int = 0, **kwargs) → tuple[int, int][source]¶

Generates a position to attack, via a chosen attack difficulty algorithm :param board: input the board to attack so the algorithm knows how large it is :param difficulty: What difficulty the AI should be (0-4) :param kwargs: Keywords arguments. ‘my_guess_board’ for difficulty ≠ 0 :return: a tuple coordinate on the grid
